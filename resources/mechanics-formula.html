力学公式速查表

本页按日本高中物理中“各种运动”相关内容整理。使用时请先判断运动模型和适用条件，再代入公式；共通考试中经常考查公式背后的物理意义、图像和实验情境。

运动学

第 1 讲
公式含义使用提示
v = Δx / Δt平均速度速度使用位移，平均速率使用路程。
a = Δv / Δt平均加速度方向改变也表示速度改变。
v = v0 + at匀加速度运动速度只适用于加速度恒定。
x = x0 + v0t + 1/2 at^2匀加速度运动位移先定正方向，重力加速度也要带符号。
v^2 - v0^2 = 2a(x - x0)不含时间的速度位移关系适合题目没有给时间时使用。
x = v0 cosθ · t
y = v0 sinθ · t - 1/2 gt^2		斜抛运动分解水平方向匀速，竖直方向匀加速度。

力与平衡

第 2 讲
公式含义使用提示
ΣF = ma牛顿第二定律ΣF 是合力。先画受力图，再分方向列式。
F = kx胡克定律x 是相对自然长的伸长或压缩量。
f ≤ μN静摩擦力范围静摩擦力不一定等于最大静摩擦力。
f' = μ'N动摩擦力方向与相对运动方向相反。
p = F / S压强液体内部常与深度和密度结合考查。
F = ρVg浮力ρ 是被排开流体的密度，V 是排开流体体积。

功和能量

第 3 讲
公式含义使用提示
W = Fs cosθ恒力做功取力在位移方向上的分量。
P = W / t
P = Fv		功率Fv 适用于力与速度同方向的瞬时功率。
K = 1/2 mv^2动能动能是标量，不带方向。
U = mgh重力势能先确定零势能面。
U = 1/2 kx^2弹性势能x 是相对自然长的形变量。
K + U = constant机械能守恒只有保守力做功时可直接使用。

动量与冲量

第 3 讲
公式含义使用提示
p = mv动量动量是矢量，方向与速度相同。
I = Ft = Δp冲量与动量变化F 为恒力时可直接用 Ft；变力看 F-t 图像面积。
m1v1 + m2v2 = m1v1' + m2v2'动量守恒系统外力冲量可忽略时成立。
e = -(v2' - v1') / (v2 - v1)反发系数一维碰撞中使用，注意速度符号。

圆周运动与单振动

第 4 讲
公式含义使用提示
v = rω线速度与角速度ω 的单位是 rad/s。
a = v^2/r = rω^2向心加速度方向始终指向圆心。
F = mv^2/r = mrω^2向心力向心力不是新力，是合力在圆心方向的效果。
T = 2π/ω周期与角速度频率 f = 1/T。
a = -ω^2x单振动加速度负号表示回复方向指向平衡位置。
T = 2π√(m/k)弹簧振子周期理想弹簧、无阻尼、小振动。
T = 2π√(l/g)单摆周期小振幅近似下成立。

万有引力

第 5 讲
公式含义使用提示
F = GmM/r^2万有引力r 是两物体质心间距离。
U = -GmM/r引力势能通常取无穷远处势能为 0。
v = √(GM/r)圆轨道速率由万有引力提供向心力推出。
T^2 = 4π^2r^3 / GM圆轨道周期对应开普勒第三定律。

刚体平衡

第 2 / 6 讲
公式含义使用提示
τ = Fl力矩l 是转轴到力的作用线的垂直距离。
τ = rF sinθ力矩大小θ 是 r 与 F 的夹角。
ΣF = 0
Στ = 0		刚体平衡条件平动平衡和转动平衡要同时成立。
xG = (m1x1 + m2x2 + ...)/(m1 + m2 + ...)质心位置一维质心公式，二维时 x、y 分别计算。

共通考试的力学题很少只考“套公式”。请把公式和图像、实验、文字条件一起复习，尤其注意守恒条件、正方向、力的来源和近似成立条件。
